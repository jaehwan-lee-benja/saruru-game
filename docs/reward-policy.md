## 🍿 팝콘 리워드 (주간 TOP3)

### 대상 · 규칙
- **주간(월~일, KST) 랭킹 TOP 3 → 팝콘 1컵 교환권.**
- 순위 확정: 매주 **일요일 24시** 마감 시점 고정.
- **유효기간 = 다음 주 월~일 (고정 1주).** 미수령 시 소멸.
  - 승자는 주 마감 후 확정 → "지난주 TOP3가 이번 주에 수령" 구조. 화면엔 정확한 날짜로 표기.

### 수령 · 검증 (핵심)
- **추천: 인게임 셀프 수령(서버 1회용 처리).** 당첨자 화면의 "직원 앞에서 [팝콘 받기]" 버튼 → 서버가 수령완료 1회 기록 → 중복수령 차단. 직원은 초록 "수령완료 ✓" 확인만.
- 표시: 닉네임 · 순위(🥇🥈🥉) · 유효기간 · 상태(미수령/수령완료).
- 대안: 6자리 교환코드 + 직원 확인 페이지.

### 인게임 안내
- 당첨자: "🎉 지난주 TOP3! 이번 주 사르르목장에서 **팝콘 받아가세요** (m/d~m/d)".
- 일반: "주간 TOP3 = 팝콘 1컵! 이번 주 순위 도전 🏆".

### 부정 · 비용
- 최대 3컵/주 → 비용 미미. 서버 치트검증(점수범위·최소시간·plausibility·레이트리밋) 기존 작동. 실물 보상이라 TOP3 지급 전 비현실 급상승 자동 플래그 권장. 교환권 = (주차×플레이어) 1회용.

### 구현 필요물 (참고)
- `game` 스키마 주간 당첨/교환 테이블(week_start, rank, player_id, nickname, claimed, claimed_at) — 공유DB DDL 게이트.
- 주 마감 top3 스냅샷(Edge Function) + 당첨 배너·받기 버튼 + (선택) 대시보드 교환권 현황.
