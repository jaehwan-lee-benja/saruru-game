# 사르르목장 게임 🐄

사르르목장 고객을 위한 웹 미니게임 사이트.

## 똥피하기 (M1)

떨어지는 💩을 피하고 🥛우유·🍀네잎클로버를 모으는 HTML5 Canvas 캐주얼 게임.

- 설치·로그인 없이 브라우저에서 바로 플레이
- 데스크톱(방향키) + 모바일(화면 터치·드래그) 지원
- 목숨 3개, 재도전, 최고점수 저장(브라우저 로컬)
- 레트로 픽셀아트: 사르르목장 젖소 마스코트(SD/치비 옆모습) · 우유팩 모자 · 볼터치 · 걷기 애니메이션
- 고득점 아이템일수록 크고 반짝임(희귀도 시각화) + 수집 파티클/플래시 이펙트
- 8bit 레트로 효과음(WebAudio 합성) + 모바일 햅틱 진동 + 음소거 토글

## 로컬 실행

정적 파일이라 별도 빌드가 없습니다.

```bash
python3 -m http.server 8080   # → http://localhost:8080
```

또는 `index.html`을 브라우저로 바로 엽니다.

## 구조

```
index.html      게임 화면 · HUD · 오버레이
css/style.css   목장 테마 스타일
js/game.js      게임 루프 · 입력 · 충돌 · 렌더
```
