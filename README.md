# 사르르목장 게임 🐄

사르르목장 고객을 위한 웹 미니게임 사이트.

## 똥피하기 (v7 · 확정 BI 적용판)

떨어지는 소똥을 피하고 우유·아이스크림을 모으는 HTML5 Canvas 캐주얼 게임.

- 설치·로그인 없이 브라우저에서 바로 플레이
- 데스크톱(방향키) + 모바일(화면 터치·드래그) 지원
- 목숨 3개, 재도전, 최고점수 저장(브라우저 로컬)
- 8bit 레트로 효과음(WebAudio 합성) + 모바일 햅틱 진동 + 음소거 토글

### 낙하물 = 똥 + 점수 2단계뿐

등급이 **크기·색·반짝임·소리·점수**로 한꺼번에 갈리게 해서, 값어치가 눈으로 바로 읽히게 했다.

| | 크기 | 생김새 | 점수 | 연출 |
|---|---|---|---|---|
| **우유팩** | 14×18 (작음) | 크림 + 네이비 밴드, 수수 | **5점** | 단음 |
| **딸기 선데이** | 24×30 (큼) | 딸기핑크 스쿱 + 하이라이트 | **20점** | 반짝임 + 3음 아르페지오 + 진동 |
| 똥 | 19×18 | 갈색(대비용 · BI 팔레트 밖) | 목숨 −1 | 진동 + 노이즈 |

> 소프트콘은 **모자(캐릭터 선택)로만** 남기고 아이템에서는 뺐다 — 3단계면 중간 등급의
> 값어치가 시각적으로 묻힌다.

### 난이도 / 캐릭터 크기

얼굴 마크는 **BI 픽셀 마스터라 축소가 금지**다(비정수 배율 금지). 그래서 캐릭터를 줄이는 대신
**월드를 넓혔다** — 마스코트는 40px 그대로 두고 가상 해상도를 160×240 → **224×336**으로.
화면 점유가 25% → 18%로 내려가 피할 여유가 생긴다(히트박스는 얼굴 코어 26px).

### 소리 · 진동

- **음소거 버튼(🔊)은 소리와 진동을 함께 끈다** — `vibe()`가 `!muted` 게이트를 공유한다.
  진동이 안 느껴지면 먼저 이 버튼 상태(`localStorage: saruru.muted`)를 확인할 것.
- 진동은 **피격**·**게임오버**·**딸기 선데이 수집**에서 발생.
- ⚠ `navigator.vibrate`는 **iOS(아이폰) Safari에 아예 없다** — 웹 표준 미지원이라 코드로 못 고친다.
  안드로이드 크롬에서만 진동한다. 데스크톱은 API는 있으나 진동 하드웨어가 없다.

### BI 적용 (v7에서 리뉴얼)

플레이어가 **사르르목장 BI 확정 마스코트 "얼굴 마크"** 자체다.

- **마스코트** — BI 원페이저의 얼굴 마크 생성기를 그대로 이식(좌표·팔레트·셰이딩 램프 동일).
  몸통 시도는 BI에서 접고 얼굴 마크로 확정된 결정을 게임도 따른다.
- **모자 3종**(우유팩·소프트콘·스쿱) = BI 마스코트 라인업 → **캐릭터 선택** + 낙하 아이템 모티프.
- **애니메이션** — BI 확정 스펙 그대로: 눈깜박 + 모자 둥둥(3초 주기 위아래 1회 왕복).
- **팔레트** — NAVY `#385088` · DEEP INK `#26365e` · SKY `#bcccea` · CREAM `#fff6e6` ·
  STRAWBERRY `#f4a9b8` · BERRY `#e07d94`.
- **그래픽 문법** — 딥잉크 방향성 외곽선 + 하드 오프셋 그림자 + 점선 룰 + 크림 위 브랜드 도트 패턴.
  BI 팔레트에 그린이 없으므로 **초원은 크림 목초지 + 브랜드 도트**로 표현한다(그린은 BI에서
  "강조 키워드" 전용색이라 대면적 사용 금지).

> ⚠ 마스코트 생성기(`faceHeadG`/`faceDetails`/`topping`)는 BI 원본
> (`saruru-game-dashboard/brand/bi-onepager.html`)과 **동일해야 한다.** 바꿔야 하면 BI 원본 먼저.

## 로컬 실행

정적 파일이라 별도 빌드가 없습니다.

```bash
python3 -m http.server 8080   # → http://localhost:8080
```

또는 `index.html`을 브라우저로 바로 엽니다.

## 구조

```
index.html      게임 화면 · HUD · 오버레이 · 모자 선택
css/style.css   BI 팔레트 · 픽셀 UI
js/game.js      BI 스프라이트 생성기 · 게임 루프 · 입력 · 충돌 · 렌더
versions/       역대 버전 스냅샷(대시보드 버전 이력에서 플레이 가능)
m2/             M2 준비물(카카오 로그인 + 랭킹) — 미적용·미배선
```
